Servicios de Digitación

Servicios administrativos a tu medida

Somos un pequeño grupo de profesionales que busca satisfacer la necesidad de servicios de digitación a particulares o PYMES. Buscamos prestar un servicio de calidad al desarrollar procesos administrativos que puedan satisfacer las necesidades del cliente.

Servicios

Digitación

Transcripción de datos a formato word o txt. Se realiza la recuperación de datos tipo texto de imágenes o documentos pdf escaneados o fotografiados, con el objetivo de entregar un documento editable.

Transcripción

Tipeo de audios en idioma español a un documento de texto editable. Los datos son ingresados sin formato específico, respetando reglas gramaticales y ortográficas básicas.

Data Entry

Captura de datos en planilla de excel. Generalmente ingreso de datos números según especificaciones del cliente. Asociado a trabajo de indexación de data para bases de datos y/o trabajos contables básicos.

Web Scrapping

Extracción de datos de sitios web para bases de datos. Ingreso de datos en planilla excel con la captura realizados de valores numérico o cadenas de texto según especificaciones del clientes.

Administrative Outservices

Trabajos administrativos teleoperados. Realización de tareas rutinarios de ingreso de datos a plataformas internas del cliente respetando los protocolos e indicaciones dadas.
